\hypertarget{md__read_me_autotoc_md23}{}\doxysection{\texorpdfstring{Abstract}{Abstract}}\label{md__read_me_autotoc_md23}
Cities function as {\bfseries{complex adaptive systems}} formed through the interplay of decentralized bottom-\/up self-\/organization and centralized top-\/down planning interventions. {\bfseries{Rogue City Designer}} is a procedural generation framework that models this duality through:\+


\begin{DoxyEnumerate}
\item {\bfseries{Tensor field axioms}} encoding top-\/down planning intent (radial, grid, organic)
\item {\bfseries{AESP-\/based district emergence}} from road network topology (Access, Exposure, Serviceability, Privacy)
\item {\bfseries{Multi-\/scale street analysis}} using persistent homology and grid quality metrics
\item {\bfseries{Layered graph support}} for bridges, tunnels, and grade-\/separated infrastructure
\end{DoxyEnumerate}

Unlike traditional square-\/grid generators, this system produces {\bfseries{coherent, believable urban environments}} ranging from rigid Manhattan-\/style grids to organic European networks, based on axiom configuration and urban complexity principles.
