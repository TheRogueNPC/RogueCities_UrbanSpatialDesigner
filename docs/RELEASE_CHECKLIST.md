# Release Checklist

## Required Gates
- All unit tests pass.
- Integration tests (`test_full_pipeline`, `test_texture_replay`) pass.
- Determinism baseline checks pass.
- Incremental parity and cancellation coverage pass.
- No new compiler warnings in release configuration.
- Documentation for new API/config ranges is updated.

## Performance Gates
- `test_generation_stage_perf` completes successfully.
- `test_generation_latency` meets metric budgets (`full_generation_ms`, `incremental_roads_ms`, `cancel_response_ms`).
- `tools/check_perf_regression.py` passes against `tests/baselines/performance_budgets_v0.10.json`.
- Performance report artifact generated by CI.
- No unexpected regression vs prior release benchmark artifacts.

## Packaging
- Version bump committed.
- Changelog updated.
- Release notes include determinism/cancellation/incremental changes.
