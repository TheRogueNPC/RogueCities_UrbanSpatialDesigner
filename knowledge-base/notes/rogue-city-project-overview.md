# Rogue City Designer Overview (Urban Spatial Generation Framework)

Rogue City Designer is a procedural city-generation platform built in C++20 that combines tensor-guided road generation, AESP-informed district behavior, and an editor/visualizer toolchain for interactive city design.

## Key Search Context
- Domain: urban procedural generation
- Inputs: axioms, tensor fields, road policies
- Outputs: roads, districts, lots, sites, visualization/export
